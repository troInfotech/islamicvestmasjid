### Container (Relative)
```css

  position: relative;  /* Creates positioning context */

}
```

### Iframe (Absolute)
```css
iframe {
  position: absolute;  /* Breaks out of flow */
  top: 0;             /* Stick to top */
  left: 0;            /* Stick to left */
  width: 100%;        /* Fill width */
  height: 100%;       /* Fill height */
}
```
